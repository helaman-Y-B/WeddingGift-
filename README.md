# WeddingGift:
    The purpose of wedding gift is to help people have a new way to give gifts to the couple that is getting married.  

# How it works?
    The website will display some items that the couple will need after the wedding. Each item will have a price and when a user clicks it, it will send them to a page where they can select a "payment option".

    The users are not buying the product, they are giving money to the couple so that they can later buy that specific product.

# The plan:

    The page will open in a home page where a wedding closed letter will be displayed. After clicking it the user will be sent to the part where they will be able to select the products.
